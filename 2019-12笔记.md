4.关于 flutter 中 `with` 是否允许内部属性同名的研究
```
void main() {
  var res = new Res();
  res.toFoo();
}

class A {
  toFoo() {
    print('foo a!');
  }
}

class B {
  toFoo() {
    print('foo b!');
  }
}

// A B 的顺序会影响 res.toFoo() 的结果
// 表明了 with 是允许同名函数的，优先级逻辑为后来居上
class Res with  A,B {}
```
